# Personal Automation Kit

![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=white) ![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?logo=github-actions&logoColor=white)

A practical collection of **Python scripts** and **GitHub Actions** for daily life automations and coding productivity. Designed to save time, run on schedule, and grow into a sponsor-supported project.

## ✨ Features
- **Daily News Digest** (SA + Tech)
- File Organizer
- Habit Tracker
- Automated Repo Reports
- CI/CD Linting & Testing
- Auto Dependency Updates

## 🚀 Quick Start
```bash
git clone https://github.com/hngzndkpzy-png/personal-automation-kit.git
cd personal-automation-kit
pip install -r requirements.txt
```

Workflows run automatically via GitHub Actions.
